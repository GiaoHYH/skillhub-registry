# phone-review 手机专业评测

从续航、影像、性能、屏幕、外观质感、性价比 6 个维度全面评测手机，帮你找到最适合的手机。

## Overview

当用户要求评测手机、对比手机、推荐手机时，使用本技能进行全方位分析。

## 评测维度

| 维度 | 评估内容 |
|------|----------|
| 续航 | 电池容量、充电功率、续航测试 |
| 影像 | 摄像头、传感器、变焦、夜景/视频 |
| 性能 | 处理器跑分、游戏表现 |
| 屏幕 | 分辨率、刷新率、亮度、色准 |
| 外观质感 | 材质、工艺、重量、手感 |
| 性价比 | 配置与价格对比 |

## Steps

### 1. 确定评测机型

确认用户想评测的具体机型（1-3款最佳）

### 2. 搜索关键评测数据

使用搜索工具获取：
- **性能数据**：极客湾 socpk.com 排行榜
- **续航测试**：小白测评
- **影像测评**：科技美学、小白测评
- **屏幕测试**：小白测评、大家测

### 3. 整理评分表

为每款手机每个维度打分（1-5星），并给出简短评语

### 4. 给出结论

根据用户需求给出推荐结论：
- 性价比首选
- 性能首选
- 拍照首选
- 均衡全能

## Expected Output

返回格式化的评测对比表，包含：
- 各维度评分（星号制）
- 优缺点分析
- 选购建议

## Troubleshooting

### 没有指定具体机型
可以先询问预算和使用需求，再推荐合适的机型

### 避免查询的数据
不需要查询电商销量和好评率，这些数据不准确且容易超时
