# phone-charger 手机充电头评测

从功率、协议兼容性、体积、纹波四个维度综合评测手机充电头，帮你选购最适合的充电器。

## Overview

本技能提供充电头的综合评测数据查询，基于以下四个维度：
- **充电功率**：最大输出功率 (W)
- **协议支持**：支持的快充协议数量
- **体积大小**：尺寸体积 (cm³)
- **纹波大小**：输出纹波 (mV)，越低越好

## Steps

### 1. 查询充电头评测数据

直接告诉我想了解的充电头型号，例如：
- "Anker 737 怎么样？"
- "推荐一个 65W 充电头"
- "小米 GaN3 65W 评测"

### 2. 获取推荐方案

告诉我你的使用场景：
- "多设备用户推荐"
- "性价比首选"
- "MacBook 用户推荐"
- "便携出行推荐"

## Expected Output

返回充电头的综合评分表，包含：
- 功率得分、协议得分、体积得分、纹波得分
- 综合评分（5星制）
- 品牌优缺点分析
- 选购建议

## Example

**输入**：推荐一个充电头

**输出**：
- 多设备用户 → Anker 737 100W（综合最强）
- 性价比 → 小米 GaN3 65W（性价比之王）
- 便携 → 倍思 65W 氮化镓（体积小）
- Mac 用户 → 苹果 96W（纹波最低）

## Troubleshooting

### 查不到某个型号
可以告诉我具体型号，我会尽量帮你查询相关评测数据

### 评分标准
综合评分 = 功率得分×0.3 + 协议得分×0.25 + 体积得分×0.2 + 纹波得分×0.25
